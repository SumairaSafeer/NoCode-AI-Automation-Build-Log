# No-Code AI Automation: Google Form → Groq AI (Make.com)

**Build Log** — Connecting a real workflow with Google Forms, Google Sheets, Groq (Llama-3.3-70b), and Make.com.
** By: Sumaira Safeer**

## Overview
This automation:
- Triggers on new Google Form submissions.
- Sends data to Groq AI for summarization + categorization.
- Logs results and prepares for final actions (email/Slack/sheet update).

Built entirely on **Make.com free tier**.

## Build Steps
**Step 1**: Google Sheets Trigger + Groq Module.

**Step 2**: Groq API Connection.

**Step 3**: System Prompt (Agent Persona). 

**Step 4**: Dynamic User Message Mapping.

**Status**: Trigger & AI step working → Execution log shows successful runs.

## Key Technologies
- **Trigger**: Google Forms → Google Sheets.
- **AI**: Groq (llama-3.3-70b-versatile).
- **Orchestration**: Make.com.
- **Final Action**: Planned (Email / Slack / Sheet update).

## Learnings
- Make.com is great for multi-step no-code scenarios.
- Groq offers fast & cheap inference for Llama models.
- Dynamic field mapping makes automations robust.
